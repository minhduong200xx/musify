Requirement:
flutter sdk: ">=2.17.1 <3.0.0"
dart

How to run: 

```
git clone github.com/minhduong200xx/musify
```
After installed, go into your workspace and run:
```
flutter pub get
```
or 
```
flutter pub outdated
```
Last, run:
```
flutter run
```
